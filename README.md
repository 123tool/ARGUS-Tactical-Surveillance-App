# ARGUS-Tactical-Surveillance-App
APP dasbor pengawasan taktis berkinerja tinggi yang dirancang untuk mengumpulkan, memproses, dan memvisualisasikan data spasial dari umpan kamera data terbuka secara global. Alat ini memfasilitasi pemantauan waktu nyata berskala besar terhadap **95.838++ node kamera** yang tersebar di jalan raya, landmark, dan pusat kota di seluruh dunia.
